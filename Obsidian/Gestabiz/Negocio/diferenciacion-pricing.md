# Diferenciación de Pricing — Gestabiz vs. Competencia LATAM

## Tesis Central

Gestabiz no compite en **precio**, sino en **valor/completitud**. El cliente no paga menos porque Gestabiz sea barato; paga porque integra 5-6 herramientas en una, ahorrando dinero y fricción operativa.

---

## Análisis Competitivo — Pricing y Completitud

### Calendly (Global SaaS)
- **Precio base**: $120 USD/mes (~380k COP)
- **Qué hace**: Solo citas y scheduling
- **Para usar bien**: + Stripe/Square (pagos) + Slack (notificaciones) + Google Calendar (sync) + Zapier (automatización)
- **Costo real**: $120 USD + integración (60-100 USD) = $180-220 USD/mes (~575-700k COP)

**Problema para PyME Colombia**: Paga en dólares, y solo es citas. Necesita más herramientas.

---

### Calendly + Alternatives Stack (Typical LATAM Setup)
Cliente típico de salón en Bogotá busca:
1. **Citas**: Calendly o Zoho Bookings ($120-150 USD)
2. **Facturación/Contabilidad**: Wave (gratis pero incompleto) o QuickBooks ($25 USD)
3. **Empleados/Nómina**: Rippling ($8 USD x persona) = mínimo $80 USD para 10 empleados
4. **Chat/Mensajería**: WhatsApp Business o Intercom ($50 USD)
5. **Reviews**: Google Reviews (gratis) + Trustpilot (gratis)

**Stack Total**: $120 + $25 + $80 + $50 = $275 USD/mes (~880k COP)
**Pain Points**: 5 plataformas, 5 logins, 5 integraciones, datos dispersos

---

### Booksy (LATAM Leader)
- **Precio base**: $40-100 USD/mes (~130-320k COP)
- **Qué hace**: Citas + reviews + marketplace de negocios
- **Coverage**: Argentina, Brasil, Chile, Colombia, México
- **Ventaja**: Presencia LATAM fuerte, tómela referidos desde marketplace
- **Limitaciones**:
  - No tiene contabilidad integrada
  - Gestión de empleados básica (sin jerarquía, sin ausencias aprobadas)
  - Chat rudimentario
  - No tiene reclutamiento
  - No tiene recursos físicos (solo profesionales)

**Stack complementario**: Booksy + Wave/Rippling = $90-150 USD

---

### Fresha (LATAM + EU Leader)
- **Precio base**: $80-150 USD/mes (~256-480k COP)
- **Qué hace**: Citas + POS + reviews + empleados + WhatsApp
- **Coverage**: Fuerte en EU, creciente en LATAM
- **Ventaja**: POS integrado (punto de venta físico)
- **Limitaciones**:
  - Contabilidad NO integrada (solo POS básico, sin fiscalidad colombiana)
  - Chat solo WhatsApp
  - Sin reclutamiento
  - Configuración compeja (más de 50 pasos para onboarding)

**Stack complementario**: Fresha + Wave = $120-180 USD

---

### Gestabiz (Gestabiz)
- **Precio base**: 90k COP/mes (~$29 USD) — **Plan Inicio**
- **Qué hace**: Citas + empleados + ausencias + contabilidad (IVA, ICA, Retención) + chat + reclutamiento + reviews + recursos + reportes + integración Google Calendar + API abierta
- **Coverage**: Colombia native, expandiendo LATAM
- **Ventajas**:
  - Precio local (COP, no USD)
  - Impuestos colombianos integrados (IVA, ICA, Retención en la Fuente)
  - Ausencias aprobadas (exclusivo, crítico para PyME)
  - Jerarquía de empleados (manager, professional, receptionist, etc.)
  - Recursos físicos soportados (no solo profesionales)
  - Reclutamiento integrado (sin pagar LinkedIn Recruiter)
  - Chat multicanal (in-app + WhatsApp + SMS + Email)
  - Mejor UX (feedback: "muy completa y fácil de usar")

## Propuesta de Valor — ROI vs. Competencia

### Cliente Salón de Belleza (10 empleadas, Bogotá)

**Escenario A: Booksy + Stack Complementario**
```
Citas (Booksy):           $50 USD/mes
Contabilidad (Wave):      $0 (but poor)
Nómina (Rippling):       $120 USD/mes (12 x $8 = $96, + $30 overhead)
Chat (WhatsApp):         $0 (manual)
Reviews (built-in):      $0
----
Total:                   $170 USD/mes = 544k COP

Fricción:
- 2 logins (Booksy + Rippling)
- Datos no sincronizados (citas de Booksy NO están en Rippling)
- Contabilidad manual (Wave no integra)
- Ausencias? Manejadas por email/Excel
- Reclutamiento? Ninguno
```

**Escenario B: Gestabiz**
```
Plan Inicio (all-in):     90k COP/mes

Incluyendo:
- Citas ✓
- Empleados + jerarquía ✓
- Ausencias aprobadas ✓
- Contabilidad (IVA, ICA) ✓
- Chat + WhatsApp ✓
- Reviews ✓
- Reclutamiento ✓
- Recursos (sillas, espejos) ✓
- Reportes financieros ✓
- 1 login, datos sincronizados ✓
```

**Ahorro**:
- Costo directo: 544k - 90k = **454k COP/mes = 5.4M COP/año**
- Tiempo ahorrado: ~4 horas/semana en administración (Excel → Gestabiz)
- Valor tiempo (salario admin): 4h x 50k/hora = 200k COP/semana = 800k COP/mes
- **Total ahorro**: 454k (dinero) + 800k (tiempo) = **1.25M COP/mes = 15M COP/año**
- **Payback**: 90k/mes cuesta menos de 1 semana de ahorro real

## Estrategia de Pricing — Por Qué 90k es Correcto

### Principios

1. **Pricing por valor, no por costo**: Gestabiz cuesta ~15-30 USD en AWS + Supabase/mes. Pero vende a 29 USD de value (90k COP). Margen: 15x.

2. **Local pricing beats global**: Calendly cuesta 120 USD (fuera de reach), Gestabiz 90k COP (accesible para PyME Colombia).

3. **Freemium tier para validación**: Plan Gratuito (1 sede, 1 empleado, 1 servicio) = conversion funnel. Free users → Inicio ($90k) → Profesional ($200k) → Empresarial (custom).

4. **Competencia justifica incremento**: Booksy ~$50, Fresha ~$100. Gestabiz a $29 (90k COP) en el rango, pero con 5x más features.

5. **Incremento progresivo**: A medida que Gestabiz agrega features (Fase 2: impuestos 2027, Fase 3: IA, Fase 4: marketplace), pricing sube. Clientes actuales congelados a $29, nuevos ven precio mayor.

## Precios Propuestos (2026-2027)

### Actuales (Apr 2026)
- **Gratuito**: 1 sede, 1 empleado, 1 servicio (no incluye contabilidad, chat, reclutamiento)
- **Inicio**: 90k COP/mes — todo (citas, empleados, ausencias, contabilidad básica, chat, reviews)
- **Profesional**: 180k COP/mes — Inicio + recursos físicos, reportes avanzados, API
- **Empresarial**: custom (>500k COP) — SSO, SLA, soporte 24/7, integraciones custom

### Transición (Jul 2026+)
Cuando clientes usen más features (reclutamiento, reportes, chat):
- **Inicio**: 90k → 110k COP
- **Profesional**: 180k → 220k COP
- **Empresarial**: custom → 50% descuento si 1-año commitment

**Política**: Clientes actuales congelados a precio histórico por 24 meses (lock-in + gratitude).

## Elasticidad de Precio — Test (Jul 2026)

Una vez con 10-20 clientes, hacer A/B test:
- Grupo A (50%): Inicio a 90k COP (actual)
- Grupo B (50%): Inicio a 120k COP (+ 33%)

**Métricas a monitorear**:
- Tasa de conversión (free → paid)
- Churn mes 1-3
- Lifetime Value

**Hipótesis**: Conversion baja <5% con 120k, pero si sube, margen mejora.
